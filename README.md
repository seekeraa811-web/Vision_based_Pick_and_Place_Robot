# Vision-Guided Pick-and-Place System

A computer vision based robotic pick-and-place system using OpenCV, camera calibration, object detection, and robot control.

## Features

- Camera calibration
- Workspace mapping
- Cube detection
- Coordinate transformation
- Robot command interface
- Automatic pick-and-place pipeline

## Technologies

- Python
- OpenCV
- NumPy
- Python UI
- Wlkata Mirobot API
